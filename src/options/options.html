My Filters

Enter custom filter rules in uBlock Origin / AdBlock Plus syntax. One rule per line. Lines starting with ! are comments.

Examples: ||ads.example.com^ — block a domain
@@||safe.example.com^ — allow (exception)
example.com##.ad-banner — hide element by CSS selector
##[class*="advertisement"] — hide generic ad elements

Sites on this list will have all blocking disabled.
